# Bike Hunter

Personal project to find used bikes for sale in Poland.

## What it does

- Scrapes bike listings from OLX.pl (Warsaw area, 75 km radius, up to 1300 PLN)
- (Coming soon) Evaluates listings with Claude AI
- (Coming soon) Sends matches to Telegram
- (Coming soon) Runs automatically on GitHub Actions every 30 minutes

## Setup

```bash
pip install -r requirements.txt
python scraper.py
```
